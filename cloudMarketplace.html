Use Cloud Marketplace to deploy a LAMP stack Verify your deployment 1. Use Cloud Marketplace to deploy a LAMP stack Search "LAMP Certified by Bitnami". in the search bar. click LAMP Certified by Bitnami. - On the LAMP page, click Launch. - Notice: If this is your first time using Compute Engine, the Compute Engine API must be initialized before you can continue. - For Zone, select the deployment zone that Qwiklabs assigned to you. My Zone: us-central1-a - Leave the remaining settings as their defaults. --Deployment name "lampstack-1" --Machine type "micro(1 shared vCPU)" --Boot disk type "Standard Persistence Disk" --Boot disk size in GB "10" --Network interfaces "default(10.128.0.0/20)" - If you are prompted to accept the GCP Marketplace Terms of Service, do so. - Then Click: deploy - Notice: If a Welcome to Deployment Manager message appears, click Close to dismiss it. - The status of the deployment appears in the console window: lampstack-1 is being deployed. When the deployment of the infrastructure is complete, the status changes to lampstack-1 has been deployed. -Notice: After the software is installed, a summary of the details for the instance, including the site address, is displayed. 2. When the deployment is complete, click the Site address link in the right pane. - Close the congratulations browser tab. - On the GCP Console, under Get started with LAMP Certified by Bitnami, click SSH. - Notice: In a new window, a secure login shell session on your virtual machine appears. - In the just-created SSH window, to change the current working directory to /opt/bitnami, execute the following command: cd /opt/bitnami - To copy the phpinfo.php script from the installation directory to a publicly accessible location under the web server document root, execute the following command: sudo sh -c 'echo "" > apache2/htdocs/phpinfo.php' - Notice: The phpinfo.php script displays your PHP configuration. It is often used to verify a new PHP installation. - To close the SSH window, execute the following command: exit - Open a new browser tab. - Type the following URL, and replace SITE_ADDRESS with the URL in the Site address field in the right pane of the lampstack page. http://SITE_ADDRESS/phpinfo.php - Notice: A summary of the PHP configuration of your server is displayed. - Close the phpinfo tab.
